## 🚀 Cómo Usar

### Instalación
```bash
cd wiki-concept
pip install -r requirements.txt
```

### Desarrollo Local
```bash
mkdocs serve
# Acceder: http://localhost:8000
```

### Compilar para Producción
```bash
mkdocs build --clean
# Genera carpeta: site/
```
