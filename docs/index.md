## 🔄 Cómo Usar Esta Wiki

### Búsqueda
Usa la **barra de búsqueda** (esquina superior derecha) para encontrar contenido específico.

### Navegación
Usa el **menú lateral** para navegar por categorías y documentos.

### Modo Oscuro
Haz clic en el **icono de bombilla** (esquina superior derecha) para cambiar entre modo claro y oscuro.

## 💡 Consejos para Colaboradores

### Agregar o Actualizar Documentos
1. Edita los archivos `.md` en la carpeta `docs/`
2. Envía un Pull Request para revisión
3. Coordina con el propietario del documento
4. Incluye cambios significativos en el changelog

### Mantener Documentos Actualizados
- Revisa documentos trimestral
- Actualiza fechas de revisión
- Reporta información desactualizada
- Sugiere mejoras en estructura
